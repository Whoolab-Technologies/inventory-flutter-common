## ğŸ’» Local Setup

### 1. Clone the repository

```bash
git clone https://github.com/Whoolab-Technologies/inventory-flutter-common.git
cd inventory-flutter-common
```

### 2. Setup Flutter environment

This project relies on **FVM** (Flutter Version Management) for version consistency.
Refer to `.fvmrc` for the exact version (currently `3.24.5`).

Using FVM:
```bash
fvm install
fvm use
```

Or ensure your global Flutter version matches the one in `.fvmrc`.

### 3. Install dependencies

```bash
fvm flutter pub get
# or
flutter pub get
```
